FAQ-WORKFLOW使用问题解答




具体技术问题请加QQ群：690266685




  • 如何部署？
  • docker pull 6996899/workflow_net_cn
  • 如何启动软件服务？
  • docker run -itd --name wf -p 1980:1980 --mac-address=f4:0f:24:33:26:39 6996899/workflow_net_cn sh -c "/opt/workflow.net.cn/start.sh"
  • 其他未安装服务的相同网络设备要如何访问系统操作界面？
  • 需要修改系统host文件，将127.0.0.1的地址指向到安装服务的网络地址。
  • 无法登录怎么办？
  • 清理浏览器缓存或者更换浏览器之后再尝试。

当前仅支持使用Docker安装

安装步骤：
拉取镜像
docker push 6996899/workflow_net_cn:v1.5

启动容器
docker run -itd --name wf -p 1980:1980 --mac-address=f4:0f:24:33:26:39 6996899/workflow_net_cn sh -c "/opt/workflow.net.cn/start.sh"

注意 端口号可以修改成任意的，但是 mac地址必须保持f4:0f:24:33:26:39，否则将无法启动服务。

进入容器：
docker exec -it wf bash

cd /opt/workflow.net.cn/

启动服务:
/opt/workflow.net.cn/start.sh

访问WEB管理界面
修改hosts文件
linux下编辑： /etc/hosts
windows 下编辑： C:/windows/system32/drivers/etc/hosts

增加一行：
127.0.0.1 workflow.net.cn

局域网内其它客户机若要访问，请将127.0.0.1 更换成docker所在的宿主机IP地址。

web登录默认账号：000001 密码：123456

若无法登录，请参考业务库中的 person_user 表数据。
